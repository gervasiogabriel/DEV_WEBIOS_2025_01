Em busca da Web (Uma historia de amor e pitufos)

Era uma vez em tão tão disntante havia uma pitufa... desculpe... Havia um tribu de pitufos.... que sonhavam em ser programadores Vulgo Devs Web.

Um belo dia andando pelo bosque da cidade pitalandia, eles encontraram uma cobra chamada python que para os intimos era chamada de Py e ela ofereceu um fruto proibido chamado Java e o pitufinho disse NÃO eu vou jogar um CSS

Mas logo ele mudou de ideia e acabou comendo o fruto proibido e acabou conhecendo o capitão Java

Após ele conhecer o capitão Java ele decidiu programar por 40 dias e 40 noites, ele foi expulso do VsCode pelo HTML, quando ele achou que a tempestade

havia acabado veio um grande diluvio chamado Git e devastou todo o seu conhecimento pela parede e veio a chuva orte e o derrubou... Tragico FIM para pitufinho
